# DinoGames

Official website, support resources, and privacy policy for the **DinoGames** iOS app.

## About

DinoGames is an educational collection of prehistoric-themed mini-games designed to make learning fun for dinosaur enthusiasts of all ages.

Version 1.0 includes **36 interactive mini-games** featuring:

* Dinosaurs
* Pterosaurs
* Marine Reptiles

Players learn through matching games, image puzzles, and comparisons while discovering prehistoric animals, geological formations, habitats, diets, fossil footprints, eggs, teeth, flora, and fauna.

## Privacy Policy

DinoGames was intentionally designed to work without:

* User accounts
* Advertising
* Analytics
* In-app purchases
* Internet connectivity during gameplay

Player progress is stored locally on the device and is never transmitted to external servers.
